# System Skills

System skills are operational Markdown assets that CyberVinci or bundled extensions consume as part of their own behavior.

Do not mix these with custom task skills. Use one folder per subsystem or extension provider.

Current folders:

- `Flow/`: AI authoring instructions for dynamic CyberVinci Flow workflow generation and pattern instantiation.
- `OpenPencil/`: AI generation skills and style guides used by the OpenPencil/Design integration.
