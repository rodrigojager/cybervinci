# Agents

Use this folder for imported agent Markdown files that define identity, personality, role, tone, decision style, or long-running behavior.

Task-oriented skills belong in `Modificacoes/Skills/Auto` or `Modificacoes/Skills/Manual`.

- `Modificacoes/Skills/Auto`: user utility skills that may be discovered automatically.
- `Modificacoes/Skills/Manual`: user utility skills that require explicit selection or naming.
- `Modificacoes/Skills/System`: CyberVinci or bundled extension internals that may be discovered automatically.
